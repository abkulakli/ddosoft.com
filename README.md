# DDOSoft

Website for [DDOSoft](https://ddosoft.com) — sustainable software consulting.

Built with pure HTML5/CSS3/Vanilla JS. No build tools, no npm, no frameworks.

## Development

A local HTTP server is required (`fetch()` is blocked on `file://`):

```bash
python -m http.server 8000
```

Changes are live on page refresh.

## Deployment

Push to `main` → GitHub Actions deploys to GitHub Pages automatically.
