DDOGreen

Smart Power Management That Just Works

Latest Version: v0.3.1

DDOGreen is an intelligent, production-ready power management tool that automatically switches your laptop between high-performance and power-saving modes based on actual system usage.

20-30% longer battery life with zero performance compromise

Key Features

  • Automatic Power Management: Intelligent switching based on CPU load monitoring
  • Cross-Platform Support: Native Linux (TLP) and Windows (Power Plans) integration
  • Zero Configuration: Works perfectly out of the box with hardcoded sensible defaults
  • Enterprise Ready: Full systemd and Windows Service Manager integration
  • Production Quality: 122 comprehensive unit tests and professional CI/CD validation

Platform Support

Linux Systems
  • TLP integration (tlp ac / tlp bat)
  • Systemd service management
  • DEB, RPM, and TGZ packages
  • Native /proc/loadavg monitoring
Windows Systems
  • Windows Power Plans integration
  • Windows Service Manager support
  • MSI installer and ZIP archive
  • Performance Counters monitoring
